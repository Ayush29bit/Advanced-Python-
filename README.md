# Advanced Python 🚀

A production-oriented Python repository focused on writing **clean, optimized, scalable, and modern Python code** using real-world concepts and modules commonly used in backend systems, AI/ML infrastructure, and large-scale applications.

This repository is not just about Python basics but to underastand **how Python is actually written in professional environments**.

  - classes & objects

- 📋 **Logging**

- 🧱 **Python Fundamentals**
  - data types

  - clean coding practices
